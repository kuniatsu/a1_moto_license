# A1 Motorcycle License Test Data

ベトナムのA1バイク免許試験問題データとHTML解析ツールのリポジトリです。

## 📋 概要

このリポジトリには、ベトナムのA1バイク免許試験に関する以下のデータとツールが含まれています：

- **250問の試験問題データ**（JSON形式）
- **20問の必須問題データ**（重要問題）
- **HTML解析スクリプト**（Python）
- **元のHTMLファイル**

## 📁 ファイル構成

```
a1_moto_license/
├── a1Text.json          # 250問の全問題データ
├── a1Text2.json         # 20問の必須問題データ
├── parse_html.py        # HTML解析スクリプト（元版）
├── parse_html2.py       # HTML解析スクリプト（index2.html用）
├── index.html           # 元のHTMLファイル（250問）
├── index2.html          # 必須問題のHTMLファイル（20問）
├── 指示書.txt           # 作業指示書
└── README.md            # このファイル
```

## 📊 データ構造

### a1Text.json（全問題データ）
各問題には以下の情報が含まれています：

```json
{
  "question_number": 1,
  "question_text": "Câu 1: 問題文...",
  "options": [
    "選択肢1",
    "選択肢2 [正解]",
    "選択肢3"
  ],
  "option_count": 3,
  "correct_answer_numbers": [2],
  "Required": true
}
```

### a1Text2.json（必須問題データ）
20問の重要な問題（Required=true）のみを含むデータセットです。

## 🛠️ 使用方法

### HTML解析スクリプトの実行

```bash
# 全問題の解析（index.html → a1Text.json）
python3 parse_html.py

# 必須問題の解析（index2.html → a1Text2.json）
python3 parse_html2.py
```

### データの特徴

- **総問題数**: 250問
- **必須問題**: 20問（Required=true）
- **通常問題**: 230問（Required=false）
- **言語**: ベトナム語
- **正解マーク**: `[正解]`付き
- **エンコーディング**: UTF-8

## 📝 問題の種類

### 必須問題（Required=true）
- 問題番号1-20
- 特に重要な交通ルールや安全に関する問題
- 試験で必ず出題される可能性が高い問題

### 通常問題（Required=false）
- 問題番号21-250
- 一般的な交通ルールや標識に関する問題
- 試験の出題範囲に含まれる問題

## 🔧 技術仕様

- **言語**: Python 3
- **データ形式**: JSON
- **HTML解析**: 正規表現ベース
- **文字エンコーディング**: UTF-8

問題は以下のカテゴリに分類されます：
